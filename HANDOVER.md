## Как протестировать -- пошагово

### 1. Просмотр витрины (как видит покупатель)

- Откройте https://sunstore.vercel.app
- Прокрутите каталог товаров на главной странице
- У каждого товара две кнопки: **В корзину** и **Подробнее**
- Нажмите **Подробнее** -- откроется страница товара с характеристиками, ценой и кнопкой добавления в корзину

### 2. Каталог с фильтрами

- Перейдите на https://sunstore.vercel.app/array
- Вверху страницы -- фильтр по категориям (Панели, Инверторы, Аккумуляторы и т.д.)
- Переключатель валюты в шапке сайта (RUB / USD / EUR)

### 3. Корзина

- Добавьте 1-2 товара кнопкой **В корзину**
- В шапке сайта появится счётчик товаров в корзине
- Нажмите на корзину -- перейдёте на страницу /velocity
- Там можно изменить количество (+/-), удалить товар, увидеть итоговую сумму
- Доставка бесплатно при заказе от 50 000 руб.

### 4. Оформление заказа

- На странице корзины нажмите **Оформить заказ**
- Заполните: имя, email, телефон, адрес доставки
- Выберите способ оплаты (по умолчанию -- Демо-режим)
- Нажмите **Создать заказ и перейти к оплате**
- На странице оплаты выберите **Имитировать успешную оплату**
- Заказ будет помечен как оплаченный, вы увидите страницу статуса заказа

### 5. Управление категориями (админка)

1. Войдите в админку (пароль: `sunstore`)
2. В боковом меню нажмите **Категории**
3. Вы увидите 6 предустановленных категорий
4. **Добавить категорию**: нажмите кнопку **Добавить категорию** -- заполните название, slug (латиницей), описание
5. **Изменить**: нажмите **Изменить** в строке категории
6. **Удалить**: нажмите **Удалить** (с подтверждением)
7. **Порядок**: стрелки вверх/вниз меняют порядок отображения

### 6. Управление товарами (админка)

1. В боковом меню нажмите **Товары**
2. Вверху -- фильтр по категориям (нажмите на категорию для фильтра, **Все** -- сбросить)
3. **Добавить товар**:
   - Нажмите **Добавить товар**
   - Заполните название, цену, остаток, артикул (SKU)
   - **Описание** -- визуальный редактор (как в Gmail/Word): жирный, курсив, заголовки, списки, ссылки, изображения
   - **Категория** -- выберите из выпадающего списка
   - **Изображение**: нажмите **Загрузить изображение** (выберите файл с компьютера) или **Выбрать из загруженных** (выбрать из уже загруженных файлов)
   - Включите/выключите **Показывать в каталоге**
   - Нажмите **Сохранить**
4. **Редактировать**: нажмите **Изменить** в строке товара
5. **Удалить**: нажмите **Удалить** (с подтверждением)

### 7. Загрузка изображений

- В форме товара нажмите **Загрузить изображение**
- Поддерживаются: JPG, PNG, WebP, GIF, SVG (макс. 5 МБ)
- Или нажмите **Выбрать из загруженных** -- откроется галерея уже загруженных файлов
- Можно удалить загруженный файл (красная кнопка X на превью)
- Также можно вставить прямую ссылку на изображение в поле **или введите URL**

### 8. Заказы (админка)

1. В боковом меню нажмите **Заказы**
2. Вы увидите все заказы (новые сверху)
3. Нажмите на заказ для просмотра деталей
4. Измените статус: Ожидает --> Оплачен --> Отправлен --> Доставлен (или Отменён)

### 9. Настройки платежей (админка)

1. В боковом меню нажмите **Платежи**
2. Вы увидите 6 платёжных шлюзов: Т-Банк, ЮKassa, Сбербанк, CloudPayments, Робокасса, Демо-режим
3. Для подключения реального шлюза:
   - Включите переключатель
   - Выберите режим (Песочница / Боевой)
   - Введите учётные данные (Terminal Key, Secret и т.д.)
   - Нажмите **Проверить подключение**
   - Нажмите **Сделать активным**
   - Нажмите **Сохранить**
4. Без merchant-ключей магазин работает в **Демо-режиме** (имитация оплаты)

## Страницы сайта

| Страница | URL | Описание |
|---|---|---|
| Главная | `/` | Hero-секция, каталог, преимущества |
| Каталог | `/array` | Полный каталог с фильтрами по категориям |
| Конфигуратор | `/blueprint` | Собрать солнечную систему комплектами |
| Корзина | `/velocity` | Товары в корзине, управление количеством |
| Оформление | `/checkout` | Данные клиента, выбор оплаты |
| Карта товара | `/product/[id]` | Детали товара, характеристики, фото |
| Статус заказа | `/status` | Проверка статуса по номеру заказа |
| Админка | `/admin` | Dashboard со статистикой |
| Товары (админка) | `/admin/products` | Управление товарами |
| Категории (админка) | `/admin/categories` | Управление категориями |
| Заказы (админка) | `/admin/orders` | Управление заказами |
| Платежи (админка) | `/admin/settings/payments` | Настройка платёжных шлюзов |

## Важно знать

- **Данные хранятся в оперативной памяти** -- при перезапуске сервера Vercel заказы и изменения товаров могут сброситься. Для production нужно подключить базу данных (PostgreSQL).
- **Пароль админки по умолчанию**: `sunstore` -- рекомендуется сменить (установите переменную окружения `ADMIN_PASSWORD` в настройках Vercel).
- **Демо-режим оплаты** включён по умолчанию -- магазин полностью функционален для тестирования без подключённых платёжных систем.
- **Загруженные изображения** сохраняются в папке `public/uploads/` -- при redeploy на Vercel они могут очищаться. Для production рекомендуется использовать облачное хранилище (S3, Cloudflare R2).
